## ‚ú® Key Features

This portfolio was built from scratch without any heavy frameworks, focusing on pure DOM manipulation, CSS animations, and interactive math.

* **System Boot Preloader:** A custom terminal-style boot sequence that initializes before revealing the site.
* **Matrix Digital Rain:** An always-on, customized `<canvas>` background running a highly optimized Matrix falling-code effect.
* **Custom Tracking Cursor:** A glowing teal dot with a trailing outer ring that expands when hovering over interactive elements.
* **3D Glassmorphism Tilt:** Profile and project cards calculate mouse position to physically tilt in 3D space (`perspective`, `rotateX`, `rotateY`) while casting dynamic shadows.
* **Scroll-Tracking Timeline:** A glowing neon line that lights up the "Education" and "Experience" nodes as the user scrolls down the page.
* **Cyberpunk Glitch Effect:** Pure CSS text-glitch animation on the main header.
* **Mermaid.js Integration:** Renders live, responsive system architecture flowcharts directly in the browser.
